Década de 1960

Guerra Fria



Temendo um ataque russo às bases militares, os EUA criam um modelo de troca de informações decentralizada.


A ARPANET é criada pela ARPA (Advanced Research Projects Agency).


Era utilizada exclusivamente pelo Departamento de Defesa dos EUA.


Em 29 de Outubro de 1969 ocorreu a transmissão do que pode ser considerado o primeiro E-mail da história.

Década de 1970



Tensão entre União Soviética e EUA diminui.


O governo permite que pesquisadores de algumas universidades entrem na rede.


Devido a dificuldades de gerenciar a rede o governo divide-a em duas: a MILNET e a ARPANET, o que acelerou o desenvolvimento.


A partir de então a rede começou a crescer mais livre.

WEB 1.0?



Não existe nenhuma definição consolidada sobre o que é a Web 1.0


No entanto ela é considerada como a Web estática


Seus conteúdos não podem ser alterados pelos usuários, tornando-a assim como de somente leitura.

HTML



O HTML é uma liguagem de marcação criada pelo físico Tim Berners-Lee para resolver osproblemas de comunicação e disseminação de informações.


O HTML é o responsável hoje pela estrutura do texto, antes da criação do CSS ele era usado também para a formatação.


Isso é um título


Isso é parágrafo

O CSS



O CSS é uma linguagem de folhas de estilos criado para separar a formatação do conteúdo do HTML


Sua sintaxe funciona fazendo declarações nos elementos do html


.texto {
   background-color: #000;
   color: white;
   margin-top: 4px;
}
